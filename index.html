Project

This is a project plan to be used for the implementation of Theia, an application to help unsighted individuals to navigate indoors, from one spot to another. The software, developed using Android Studio, will execute the functionality to satisfy the requirements of the system.
The high level functionality of the system is:
- Getting the user's current location and final destination.
- Generating the best route for the user to safely travel to the desired location.
- Accepting user’s routes preferences
- Voice directions (e.g., 2 minutes to reach a turning point, 30 steps to take)
- Detecting obstacles and telling the user what to do in order to avoid collision
- Placing emergency calls and messages, possibly after detecting a fall or when the system cannot determine user’s situation
- Figuring out what would be the next action(s), based on the user’s schedule or habit
- Suggesting and accepting the user’s choice
